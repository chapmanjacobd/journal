I made a small fish shell function yesterday which I'd like to share:

function abbr_random_line -a name -a path
    eval "function __abbr_random_line_$name; shuf -n1 $path; end"
    abbr --add "$name" --function __abbr_random_line_$name
end

This allows me to put a text file somewhere filled with interesting commands to run. Then I can type a short command and it will substitute the commandline with one of the lines from that file.

For example, I have abbr_random_line wt ~/watch in my fish config. When I type wt and press space it will expand to a random line from the ~/watch file.
