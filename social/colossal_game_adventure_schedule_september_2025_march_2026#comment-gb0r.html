I recently set up an emulator for PS2 to check out the games in Recursive Repertoires. This site was really helpful for setting up:

https://wiki.batocera.org/systems:ps2

For Fedora Linux users: The version of PCSX2 in rpmfusion is really old and doesn't support the high compression CHD format (that you can make with chdman of mame-tools) so I recommend just downloading the AppImage instead.

Also for Wind Waker it looks like having a recent version of Dolphin will help if you want to use the GBA integrated emulator

Also also, this page is a great resource if you want to check that the BIOS was dumped correctly: https://emulation.gametechwiki.com/index.php/File_hashes
