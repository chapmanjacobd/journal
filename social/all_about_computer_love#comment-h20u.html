This is an interactive, console.log essay about love for computers, combating escapism and deactivating healing fantasies to face the world. it's only accesible through a desktop computer.

Please open your browser's developer tools (quick how-to here) and navigate to the console log. Both the console log and this window should both be visible to be able to fully read the essay as intended.
